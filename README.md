## üìå Project Overview
This project focuses on analyzing the performance, risk exposure, and return efficiency of Exchange-Traded Funds (ETFs) over a 5-year period using Power BI.

The objective of this analysis is to transform financial market data into actionable insights that help evaluate investment performance, understand volatility patterns, and support data-driven portfolio decisions.

The dashboard combines financial KPIs, risk analysis, and time-based performance evaluation to simulate a real-world investment analytics reporting environment.

- Understand seasonal performance trends using monthly analysis.
- Provide executive-level insights for investment decision-making.
